# Projeto Limpeza
Um script em Bash descomplicado, com um menu interativo, concebido para automatizar a limpeza e organização do seu computador.

Ele auxilia na manutenção da sua pasta de Downloads, eliminando ficheiros temporários e duplicados, além de fornecer informações sobre o estado do sistema.

Tudo consolidado num único arquivo, tornando a execução incrivelmente simples.
